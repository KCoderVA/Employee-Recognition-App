Power Automate Modular Approval Table Preview

New Modular Approach: Your table is now built using individual @{concat(...)} expressions that you can mix and match. Each expression automatically omits rows when values are "None provided", empty, or null. You manually control line breaks in Power Automate.

Modular Benefits:
  • Flexibility: Pick and choose exactly which fields to include
  • Reusability: Same expressions work across different approval types
  • Clean Logic: Automatic omission of empty/null/"None provided" values
  • Manual Control: You decide line breaks and formatting in Power Automate
  • Easy Testing: Test individual expressions independently
  • Maintainability: Update one expression to change all instances

Implementation Guide

How to Use Your Modular Expressions:

1. Start with table header: Always include the header expression first
2. Add desired rows: Copy/paste any combination of row expressions
3. Manual line breaks: Add line breaks between expressions in Power Automate
4. Test individually: Each expression can be tested independently
5. Dynamic content: Rows automatically hide when data is missing
